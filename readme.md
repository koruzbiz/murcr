# What is MurCr?

MurCr is a free application that allows you to process audio/video files and produce different types of output.  
The application has been developed by Körüz.biz, a civil society organization operating in Turkiye since 2014.

# What can you do with MurCr?

With the application, you can perform the following tasks:

- You can extract one or more time ranges (segments) from your audio/video files, or create a new video by merging selected time ranges sequentially.
- You can convert audio files into videos using any image you choose, or disable the video or audio output of an existing video.
- You can merge audio/video files located in a directory in a single operation and produce output with different video or audio properties.
- You can convert your audio/video files into different formats, define the output quality, and limit the output file size in MB.
- You can split your audio/video files into equal parts based on time duration or file count.

# Safe and controlled

MurCr assists you with smart tools during the audio/video processing and creation process.

- **Protection mechanisms:** MurCr always works on a copy of the file you provide. After the files are copied into the project directory, all operations are performed on the copied files.

- **Video resolution conflict detection:** MurCr analyzes the files you want to create or merge, detects differences in video resolution values, and determines the primary video resolution to be used in the output file. It also offers different options to prepare the most suitable visual configuration.

- **Timeline overlap detection:** Another analysis detects overlapping time ranges originating from the same source. If you choose, these time ranges can be automatically merged.

- **Normalization checks:** MurCr normalizes your files to minimize deviations caused by different video resolutions or FPS values. If a deviation greater than 0.2 seconds is detected, fast trimming tools are not used. This helps prevent visual issues such as FPS jumps or frame shifts.

- **Stereo output:** MurCr generates stereo output for all tasks except splitting. Stereo channels are preserved for stereo sources. For mono sources, the active channel is duplicated to the other channel. Low-frequency audio is also upsampled to 48,000 Hz to produce stereo output files.

- **Reports:** With the integrated reporting tools, you can analyze the file you are working on or any file within the project directory based on criteria such as size, video resolution, FPS, and stereo/mono. The directory report can be exported to HTML format and reviewed easily using screen reader shortcuts.

# Flexible structure and advanced accessibility features

MurCr offers a satisfying user experience with its flexible workflow and accessibility features.

- **Simple workflow:** You can mark the areas to be trimmed in your files by pressing a single key.

- **Your choice:** You can work with a single file or with multiple files.

- **Customizable shortcuts:** The entire MurCr interface can be controlled with keyboard shortcuts. You can change the shortcuts as you wish or reset them later.

- **Advanced accessibility:** MurCr provides a screen reader focus management feature through its custom shortcut engine. For example, while the "Alt+E" shortcut triggers the "Complete Task" action, pressing "Alt+K" followed immediately by "Alt+E" moves the screen reader focus directly to the "Complete Task" button. This allows you to navigate directly to interface elements without using the Tab key.

- **Smart window management:** If MurCr detects a window appearing while a file is playing, playback is paused so that the screen reader message can be clearly understood. Once the window is closed, playback resumes from where it stopped, and the screen reader is redirected to an empty area to prevent unnecessary speech.

- **Project directory:** You can name the project you create, move it to another directory, and reopen or modify the same project even after a long time. When the task is completed, you can copy the output file or project directory to the clipboard, delete it, or use it in a new task.

- **Built-in add-on:** You can enable the NVDA add-on from the settings and send files directly to MurCr using the NVDA+ALT+K shortcut.

- **Multi-language support:** The MurCr interface is available in 11 different languages.
