# Was ist MurCr?

MurCr ist eine kostenlose Anwendung, mit der Sie Audio/Video-Dateien verarbeiten und unterschiedliche Ergebnisse erzeugen können.  
Die Anwendung wurde von Körüz.biz entwickelt, einer zivilgesellschaftlichen Organisation, die seit 2014 in Turkiye tätig ist.

# Was können Sie mit MurCr tun?

Mit der Anwendung können Sie folgende Aufgaben ausführen:

- Sie können einen oder mehrere Zeitbereiche (Segmente) aus Ihren Audio/Video-Dateien extrahieren oder eine neue Videodatei erstellen, indem Sie ausgewählte Zeitbereiche nacheinander zusammenführen.
- Sie können Audiodateien mithilfe eines beliebigen Bildes in Videos umwandeln oder die Video- bzw. Audioausgabe eines vorhandenen Videos deaktivieren.
- Sie können Audio/Video-Dateien in einem Verzeichnis in einem einzigen Vorgang zusammenführen und eine Ausgabe mit unterschiedlichen Video- oder Audioeigenschaften erzeugen.
- Sie können Ihre Audio/Video-Dateien in verschiedene Formate konvertieren, die Ausgabequalität festlegen und die Größe der Ausgabedatei in MB begrenzen.
- Sie können Ihre Audio/Video-Dateien anhand der Zeitdauer oder der Anzahl der Dateien in gleich große Teile aufteilen.

# Sicher und kontrolliert

MurCr unterstützt Sie während der Audio/Video-Verarbeitung und -Erstellung mit intelligenten Werkzeugen.

- **Schutzmechanismen:** MurCr arbeitet immer mit einer Kopie der von Ihnen bereitgestellten Datei. Nachdem die Dateien in das Projektverzeichnis kopiert wurden, werden alle Vorgänge mit den kopierten Dateien durchgeführt.

- **Erkennung von Videoauflösungskonflikten:** MurCr analysiert die Dateien, die Sie erstellen oder zusammenführen möchten, erkennt Unterschiede in den Videoauflösungen und bestimmt die primäre Videoauflösung für die Ausgabedatei.

- **Erkennung von Zeitachsenüberschneidungen:** Eine weitere Analyse erkennt überlappende Zeitbereiche aus derselben Quelle. Auf Wunsch können diese Zeitbereiche automatisch zusammengeführt werden.

- **Normalisierungsprüfungen:** MurCr normalisiert Ihre Dateien, um Abweichungen durch unterschiedliche Videoauflösungen oder FPS-Werte zu minimieren. Wird eine Abweichung von mehr als 0,2 Sekunden erkannt, werden schnelle Schnittwerkzeuge nicht verwendet.

- **Stereo-Ausgabe:** MurCr erzeugt für alle Aufgaben außer dem Aufteilen eine Stereo-Ausgabe. Stereo-Kanäle bleiben bei Stereoquellen erhalten. Bei Monoquellen wird der aktive Kanal auf den anderen Kanal kopiert.

- **Berichte:** Mit den integrierten Berichtswerkzeugen können Sie Dateien nach Kriterien wie Größe, Videoauflösung, FPS und Stereo/Mono analysieren. Der Verzeichnisbericht kann im HTML-Format exportiert und mit Screenreader-Kurzbefehlen überprüft werden.

# Flexible Struktur und erweiterte Barrierefreiheitsfunktionen

MurCr bietet dank seines flexiblen Arbeitsablaufs und seiner Barrierefreiheitsfunktionen eine komfortable Benutzererfahrung.

- **Einfacher Arbeitsablauf:** Sie können die zu schneidenden Bereiche Ihrer Dateien durch Drücken einer einzigen Taste markieren.

- **Ihre Wahl:** Sie können mit einer einzelnen Datei oder mit mehreren Dateien arbeiten.

- **Anpassbare Tastenkombinationen:** Die gesamte MurCr-Oberfläche kann mit Tastenkombinationen gesteuert werden.

- **Erweiterte Barrierefreiheit:** MurCr bietet eine Screenreader-Fokusverwaltung durch seine eigene Shortcut-Engine.

- **Intelligente Fensterverwaltung:** Wenn während der Wiedergabe ein neues Fenster erkannt wird, wird die Wiedergabe pausiert.

- **Projektverzeichnis:** Sie können Projekte speichern, verschieben und später erneut bearbeiten.

- **Integriertes Add-on:** Dateien können mit der Tastenkombination NVDA+ALT+K direkt an MurCr gesendet werden.

- **Mehrsprachige Unterstützung:** Die MurCr-Oberfläche ist in 11 verschiedenen Sprachen verfügbar.
