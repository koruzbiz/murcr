# Что такое MurCr?

MurCr — это бесплатное приложение, позволяющее обрабатывать аудио/видео файлы и получать различные результаты.  
Приложение разработано Körüz.biz — организацией гражданского общества, работающей в Turkiye с 2014 года.

# Что можно делать с MurCr?

С помощью приложения вы можете выполнять следующие задачи:

- Извлекать один или несколько временных интервалов (сегментов) из аудио/видео файлов или создавать новое видео, объединяя выбранные интервалы последовательно.
- Преобразовывать аудиофайлы в видео, используя любое изображение, или отключать видео- или аудиовыход существующего видео.
- Объединять аудио/видео файлы из одной папки за одну операцию и получать результат с различными параметрами аудио или видео.
- Конвертировать аудио/видео файлы в различные форматы, задавать качество выходного файла и ограничивать его размер в MB.
- Делить аудио/видео файлы на равные части по длительности или по количеству файлов.

# Безопасно и под контролем

MurCr помогает вам интеллектуальными инструментами во время обработки и создания аудио/видео файлов.

- **Механизмы защиты:** MurCr всегда работает с копией предоставленного файла. После копирования файлов в каталог проекта все операции выполняются с копиями.

- **Обнаружение конфликтов разрешения видео:** MurCr анализирует файлы, которые вы хотите создать или объединить, выявляет различия в разрешении видео и определяет основное разрешение для выходного файла.

- **Обнаружение перекрытия на временной шкале:** Дополнительный анализ выявляет пересекающиеся временные интервалы из одного источника. При необходимости такие интервалы могут быть объединены автоматически.

- **Проверки нормализации:** MurCr нормализует файлы, чтобы минимизировать отклонения, вызванные различным разрешением видео или значениями FPS.

- **Стереовыход:** MurCr создаёт стереовыход для всех задач, кроме разделения. Стереоканалы сохраняются для стереоисточников.

- **Отчёты:** С помощью встроенных инструментов отчётов можно анализировать файлы по таким параметрам, как размер, разрешение видео, FPS и режим стерео/моно.
