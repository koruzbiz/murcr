# 什么是 MurCr？

MurCr 是一个免费的应用程序，可用于处理音频/视频文件并生成不同类型的结果。  
该应用程序由 Körüz.biz 开发，这是一个自 2014 年以来在 Turkiye 运作的公民社会组织。

# 使用 MurCr 可以做什么？

使用该应用程序，您可以执行以下任务：

- 从音频/视频文件中提取一个或多个时间段（片段），或将选定的时间段顺序合并以创建新的视频。
- 使用任意图像将音频文件转换为视频，或禁用现有视频的音频或视频输出。
- 将目录中的音频/视频文件一次性合并，并根据不同的音频或视频属性生成输出。
- 将音频/视频文件转换为不同格式，设置输出质量，并限制输出文件的大小（MB）。
- 根据时间长度或文件数量将音频/视频文件分割为相等的部分。

# 安全且可控

MurCr 在音频/视频处理和创建过程中为您提供智能工具支持。

- **保护机制：** MurCr 始终在您提供文件的副本上工作。文件复制到项目目录后，所有操作都在副本上执行。

- **视频分辨率冲突检测：** MurCr 会分析需要创建或合并的文件，检测不同的视频分辨率，并确定输出文件使用的主要分辨率。

- **时间轴重叠检测：** 另一项分析可检测来自同一来源的重叠时间段。如有需要，这些时间段可以自动合并。

- **规范化检查：** MurCr 会对文件进行规范化，以减少由于不同视频分辨率或 FPS 值造成的偏差。

- **立体声输出：** 除分割任务外，MurCr 在所有操作中都会生成立体声输出。对于立体声源会保留原始声道。

- **报告：** 通过内置报告工具，您可以根据文件大小、视频分辨率、FPS 以及立体声/单声道等条件分析文件。

# 灵活结构和高级无障碍功能

MurCr 通过灵活的结构和无障碍功能提供良好的用户体验。

- **简单操作：** 只需按下一个键即可标记文件中需要剪切的区域。

- **由您决定：** 您可以处理单个文件或多个文件。

- **可自定义快捷键：** MurCr 的整个界面都可以通过键盘快捷键进行控制。

- **高级无障碍功能：** MurCr 通过其快捷键引擎提供屏幕阅读器焦点管理功能。

- **智能窗口管理：** 如果在播放过程中检测到新窗口，播放将自动暂停。

- **项目目录：** 您可以保存项目，并在很长时间后重新编辑。

- **内置插件：** 您可以使用 NVDA+ALT+K 快捷键将文件直接发送到 MurCr。

- **多语言支持：** MurCr 界面支持 11 种语言。
