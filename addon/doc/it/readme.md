# Che cos'è MurCr?

MurCr è un'applicazione gratuita che consente di elaborare file audio/video e ottenere diversi tipi di risultati.  
L'applicazione è stata sviluppata da Körüz.biz, un'organizzazione della società civile attiva in Turkiye dal 2014.

# Cosa puoi fare con MurCr?

Con l'applicazione puoi eseguire le seguenti operazioni:

- Puoi estrarre uno o più intervalli di tempo (segmenti) dai tuoi file audio/video oppure creare un nuovo video unendo sequenzialmente intervalli selezionati.
- Puoi convertire file audio in video utilizzando qualsiasi immagine oppure disattivare l'uscita audio o video di un video esistente.
- Puoi unire file audio/video presenti in una cartella in un'unica operazione e ottenere un output con diverse proprietà audio o video.
- Puoi convertire i file audio/video in diversi formati, definire la qualità dell'output e limitare la dimensione del file di output in MB.
- Puoi dividere i file audio/video in parti uguali in base alla durata o al numero di file.

# Sicuro e controllato

MurCr ti assiste con strumenti intelligenti durante il processo di elaborazione e creazione audio/video.

- **Meccanismi di protezione:** MurCr lavora sempre su una copia del file fornito. Dopo che i file vengono copiati nella cartella del progetto, tutte le operazioni vengono eseguite sulle copie.

- **Rilevamento dei conflitti di risoluzione video:** MurCr analizza i file che desideri creare o unire, rileva le differenze nelle risoluzioni video e determina la risoluzione principale da utilizzare nel file di output.

- **Rilevamento delle sovrapposizioni nella timeline:** Un'altra analisi rileva intervalli di tempo sovrapposti provenienti dalla stessa sorgente. Se lo desideri, questi intervalli possono essere uniti automaticamente.

- **Controlli di normalizzazione:** MurCr normalizza i file per ridurre le deviazioni causate da diverse risoluzioni video o valori FPS.

- **Output stereo:** MurCr genera output stereo per tutte le operazioni tranne la divisione. I canali stereo vengono preservati per le sorgenti stereo.

- **Report:** Gli strumenti di report integrati consentono di analizzare i file secondo criteri come dimensione, risoluzione video, FPS e stereo/mono.

# Struttura flessibile e funzionalità di accessibilità avanzate

MurCr offre un'esperienza utente efficace grazie alla sua struttura flessibile e alle funzionalità di accessibilità.

- **Flusso di lavoro semplice:** Puoi contrassegnare le aree da tagliare nei file premendo un solo tasto.

- **La scelta è tua:** Puoi lavorare con un singolo file o con più file.

- **Scorciatoie personalizzabili:** L'intera interfaccia di MurCr può essere controllata tramite scorciatoie da tastiera.

- **Accessibilità avanzata:** MurCr fornisce la gestione del focus del lettore di schermo tramite il suo motore di scorciatoie.

- **Gestione intelligente delle finestre:** Se viene rilevata una finestra durante la riproduzione, la riproduzione viene messa in pausa.

- **Cartella di progetto:** Puoi salvare e modificare i progetti anche molto tempo dopo la loro creazione.

- **Add-on integrato:** Puoi inviare file direttamente a MurCr utilizzando la scorciatoia NVDA+ALT+K.

- **Supporto multilingue:** L'interfaccia MurCr è disponibile in 11 lingue diverse.
