# MurCr क्या है?

MurCr एक निःशुल्क अनुप्रयोग है जो आपको ऑडियो/वीडियो फ़ाइलों को संसाधित करने और विभिन्न प्रकार के परिणाम प्राप्त करने की अनुमति देता है।  
यह अनुप्रयोग Körüz.biz द्वारा विकसित किया गया है, जो 2014 से Turkiye में कार्यरत एक सिविल सोसाइटी संगठन है।
